# react-native-hardwarekey-event

Easily add JS callbacks to hardware KeyEvents

## Installation

```sh
npm install react-native-hardwarekey-event
```

## Usage


```js
import { multiply } from 'react-native-hardwarekey-event';

// ...

const result = multiply(3, 7);
```
